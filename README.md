# 🎓 Student Management System

A console-based Student Management System built with Python. This application allows users to manage student records through a simple menu-driven interface. It demonstrates fundamental programming concepts such as functions, dictionaries, lists, loops, and modular program design.

---

## 📖 Overview

The Student Management System is designed to simulate a basic school administration application. Users can add, update, search, and delete student records while automatically calculating grades and averages.

This project is part of my Python learning journey toward becoming a Full-Stack Developer and AI Engineer.

---

## ✨ Features

### 👨‍🎓 Student Management

* Add new students
* View all students
* Search for a student
* Update student information
* Delete student records

### 📊 Academic Records

* Store grades for multiple subjects
* Calculate average grade
* Display highest-performing student
* Display lowest-performing student
* View individual student report

### 📋 Data Management

* Display all student records
* Count total students
* Clear all student records
* Exit the application safely
